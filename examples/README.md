# Ruby examples

Pure-Ruby examples for the `base64` library as provided by
[go-embedded-ruby](https://github.com/go-embedded-ruby/ruby) (rbgo). Run them
with the `rbgo` interpreter:

```sh
rbgo examples/base64_usage.rb
```

| File | Shows |
| --- | --- |
| [`base64_usage.rb`](base64_usage.rb) | Strict, URL-safe and RFC 2045 encode/decode. |

Each example is executed as-is under rbgo (`require "base64"`).
